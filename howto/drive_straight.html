How To: Drive Straight
Class DriveBase
Method straight(distance)
Description

Drives straight for a given distance and then stops.
Paramters distance (distance: mm) – Distance to travel.
Example(s): 

# Drive forward 50 cm
robot.straight(500)
# Drive backwards 50 cm
robot.straight(500)
